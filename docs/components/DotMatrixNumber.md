# DotMatrixNumber

`DotMatrixNumber`는 숫자를 5x7 도트 매트릭스(Dot Matrix) 형태로 시각화하는 React 컴포넌트입니다. 주로 점수 표시, 카운트다운, 또는 레트로/모던 감성의 UI 요소로 사용됩니다.

## 주요 특징

- **5x7 사각형 그리드**: 각 숫자를 35개의 개별 사각형 도트로 표현하여 독특한 시각적 효과를 줍니다.
- **애니메이션 지원**: `framer-motion`을 사용하여 숫자 변경 시 각 사각형의 상태 변화가 부드럽게 전환됩니다.
- **다이내믹 컬러**: 사각형의 색상을 자유롭게 변경할 수 있어, 배경색이나 상태값에 따른 시각적 피드백이 가능합니다.
- **자동 자릿수 처리**: 숫자 `10`의 경우 자동으로 `1`과 `0` 두 자리를 나란히 표시합니다.

## 사용 방법

```tsx
import { DotMatrixNumber } from "@/shared/ui/DotMatrixNumber";

function MyComponent() {
  return (
    <DotMatrixNumber 
      value={7} 
      color="#2FB06B" 
      dotSize="md" 
    />
  );
}
```

## Props

| Prop | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `value` | `number` | **필수** | 표시할 숫자 (0~10 권장) |
| `color` | `string` | `"white"` | 활성화된 도트의 색상 |
| `dotSize` | `"sm" \| "md" \| "lg"` | `"md"` | 도트의 크기 및 간격 설정 |
| `className` | `string` | - | 전체 컨테이너에 적용할 추가 스타일 |
| `dotClassName` | `string` | - | 각 도트에 적용할 추가 스타일 |

## 도트 크기 가이드

- **sm**: 아주 작은 텍스트나 보조 지표용
- **md**: 일반적인 점수 표시 및 카드 내부용 (기본값)
- **lg**: 메인 인터랙션 영역이나 강조용 큰 숫자

## 디자인 가이드

- **배경과의 대비**: 활성화되지 않은 도트는 기본적으로 투명도(`0.05`)가 적용되어 있어, 배경색과 대비되는 `color`를 지정하는 것이 좋습니다.
- **스케일링**: 숫자 `10`이 표시될 때 가로 너비가 넓어지므로, 부모 컨테이너에 충분한 너비가 확보되어야 합니다. (내부적으로 자동으로 `scale-90`이 적용됩니다)
